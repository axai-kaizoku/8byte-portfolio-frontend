# Portfolio using yahoo-finance2 and Alpha Vantage

- Scaffolded app using create t3-app.

# [8byte-portfolio-frontend.vercel.app](https://8byte-portfolio-frontend.vercel.app/)

# TechStack

- Nextjs
- Typescript
- Tailwindcss
- React Query
- Recharts for charts
- Tanstack-table (react-table) for table

# Usage

- clone the repo
- copy .env.example to .env
- add your backend baseurl
- Run `pnpm install`
- Run `pnpm dev`
